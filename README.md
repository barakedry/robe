**Robe** wraps around [monk](https://github.com/Automattic/monk) to provide a 
simple yet effective ODM library for MongoDB. _Inspired by [mongorito](http://mongorito.com/)_.

## Building

To run the tests:

    $ npm install -g gulp
    $ npm install
    $ npm test
