# Robe

**WORK IN PROGRESS - NOT YET READY**

Inspired by [mongorito](http://mongorito.com/), 
**robe** wraps around [monk](https://github.com/Automattic/monk) to provide a 
simple yet effective ODM library for MongoDB.

## Building

To run the tests:

    $ npm install -g gulp
    $ npm install
    $ npm test
